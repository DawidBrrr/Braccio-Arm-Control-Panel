# Braccio-Arm-Control-Panel
Python-based graphical interface for controlling the Braccio robotic arm. 

<img width="958" height="924" alt="image" src="https://github.com/user-attachments/assets/87653e2b-3c14-4269-b664-fa5f9a6329b4" />

## Features
- **Python Control Panel**: A PyQt6-based GUI for controlling the arm via serial communication.
- **Arduino Firmware**: Handles servo commands and executes movements on the Braccio arm.
- **Real-Time Control**: Adjust servo positions and visualize movements instantly.

## Project Structure
- **ControlPanel/**: Python GUI application.
- **Arduino/**: Firmware for the Braccio robotic arm.
